== Description ==

MyD Delivery integration with SumUp enable SumUp Payment Gateway options to MyD Delivery Pro.

== Installation ==

1. Upload the plugin files to the WordPress
2. Activate the plugin in WordPress
3. Go to MyD Delivery Pro -> Settings -> Payment and fill out the settings
4. Done, just receive payments with SumUp payment gateway and MyD Delivery.
